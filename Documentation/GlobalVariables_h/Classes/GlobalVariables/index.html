Introduction

This class contains Global variable declaration

Discussion

It contains variable declaration that are commonly used throughout the app so that accessing and calling is very easy. Also it cointans Singleton class that contains global variables and global functions.It’s an extremely powerful way to share data between different parts of code without having to pass the data around manually.

Discussion

A singleton object provides a global point of access to the resources of its class. Singletons are used in situations where this single point of control is desirable, such as with classes that offer some general service or resource. You obtain the global instance from a singleton class through a factory method.
